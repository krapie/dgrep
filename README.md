# dgrep

> [!NOTE]
> Worker Pool Implementation will be added soon.

dgrep is a advanced grep tool for searching keywords in files or directories.

## Installation

```bash
## Clone repository
git clone https://github.com/krapie/dgrep.git

## Change directory
cd dgrep

## Build
make build
```

## Usage

```bash
./dgrep {keyword} {path}
```

## Benchmark

```bash
time ./dgrep {keyword} {path}
```
